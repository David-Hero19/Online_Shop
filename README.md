# Online Shop 🛒

A full-stack eCommerce web app built with Python (Flask) and Stripe for secure payment processing.

## Features
- Product catalog with category filtering and search
- Shopping cart with live quantity controls
- Stripe Checkout for secure payments
- Fully responsive design

## Tech Stack
- Python / Flask
- Stripe API
- HTML, CSS, JavaScript

## Setup
1. Install dependencies: `pip install flask`
2. Set your Stripe keys as environment variables
3. Run: `python app.py`
4. Visit: `http://localhost:5000`
